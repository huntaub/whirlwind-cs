Domain Name System (DNS)

Answers the question "what is the IP address for 'google.com'?"

Domain Name System

  • Hierarchical
  • 13 Root Servers

A worked example on the board for www.google.com.

It's just that easy.

  1. Type in a website.
  2. Connect over TCP to your DNS servers (port 43).
  3. DNS looks up the IP address associated with the website.
  4. Connect over TCP to the IP address returned (port 80).
  5. Send an HTTP GET request for your resource.
  6. Receive the information, display to user.
