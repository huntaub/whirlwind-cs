From Electrons to Facebook
A Whirlwind Introduction to CS

University of Virginia CS 1501

Instructor: Hunter Leath
Advisor: Dave Evans

A new single credit student taught course at UVa that aims to provide a holistic look at how computers work from the ground up without teaching students programming. The course is targeted at students who are not currently computing majors but are interested in learning more about the field.

Enroll Now

Should I take this course?

This course is geared towards motivated non-computing majors who fall into the following two categories.

  1. Students who are interested in Computing majors but want more information about what such a curriculum would consist of. In this sense, this course is meant to be a sampler of the different topics covered in most Computer Science programs.
  2. Students who are not interested in Computing majors but want to learn more about how the computers they interact with everyday actually work. This course will provide a simple, bottom-up approach to learning more about computers without having to learn (or know) how to program.

Why is this course important?

Far too many students come to Universities without an understanding of what is included in a Computer Science curriculum. This course is an attempt to solve this need without requiring that students learn the minutia of programming to appreciate each part of the computer. Additionally, at many institutions, there is no class designed for students who are not Computing majors but want to learn more about computers without taking basic programming courses.

This course will attempt to increase interest in computers from students in all educational backgrounds.

Why no programming?

For students in Computing majors, programming is a near-universal requirement because of its ability to prepare students for software engineering jobs and have total control of the devices that are being studied. While many important concepts are communicated effectively through programming (namely abstraction and recursion), the extra burden of learning the seemingly arbitrary syntax and type rules can reduce the educational value for students who do not plan to write software for a living.

This course will provide students with an opportunity to quickly get a feeling for how computers work at every level without this obstacle.

    See Curriculum: CS 2330
  • Week 3 - From Logic to State
    Clocks, Registers, Basic Processors
    See Curriculum: CS 2330
  • Week 4 - The General Processor
    Memory, Distinction between Code and Data
    See Curriculum: CS 3330
  • Week 5 - Programming in the Abstract
    What is 'programming'? How does it work?
    See Curriculum: CS 2150 / 4610
  • Week 6 - The General Program
    What if we want to run more than one program?
    See Curriculum: CS 4414
  • Week 7 - Interfacing with the User
