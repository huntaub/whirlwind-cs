Using the Files

Each of the provided Logisim files contains several Circuits which have the functions that we have examined in class. You will see a list of circuits at the top part of left pane. You can double click on the circuits in order to view and edit them.

It is important to note that Logisim has two separate modes when working with the circuits files: editing and running. In the top left of the program, you will see two important icons: one that looks like a hand and one that looks like a cursor. In order to toggle the buttons (green dots) or clock (green squiggle), you must have selected the hand icon in the top left. Similarly, in order to edit the circuit (place new gates or wires, you must have the cursor selected in the top left.

The files each have the following circuits. Be sure to go back to the slides to get information about circuits that you do not remember.

circuits.circ

This file contains the combinatoral logic that we talked about in the Boolean Algebra Lecture.

  1. sleeping - The circuit that determined (with high accuracy) whether a student was asleep or not during a lecture.
  2. chipotle - The circuit that determined whether I was eating Chipotle for dinner (or lunch? or breakfast...?).
  3. pav - The circuit that determined whether the number of people in the Five Guys line at the Pav was greater than 5.
  4. encoder - The circuit that tells you which input line is "ON".
  5. muxer - The circuit that tells you the status of one of the input lines.
  6. half-adder - A circuit that adds two one-bit numbers.
  7. full-adder - A circuit that adds two one-bit numbers, but allows a carry-over digit to come into the calculation.
  8. ripple-carry - A circuit that adds two very large numbers (8 bits / 8 wires long) by utilizing a bunch of the full-adders.
  9. sr latch - The first introduction of memory that we looked at in class.

state.circ

This file contains the simple counters that we talked about in the State Lecture.

  1. simple - A circuit that utilizes a simple Flip-Flop. Try changing the value of the flip flop by toggling the clock (the squiggly icon).
  2. counter - A circuit that counts the number of people in line at Five Guys. Remember that the value is only updated in memory when the clock is toggled. It utilizes our adder, a simple flip flop, and a comparator.
  3. counter3 - A circuit that allows the Five Guys employees to specify whether they have lots of staff today or not. This adds a muxer to the previous circuit.

Creating New Circuits

Since the purpose of this class is understanding and exploration, I encourage interested students to play around in Logisim to get a feel for the program and how the different components work. You can make a new circuit by clicking the green plus icon in the top left of the screen. After naming it, you can place components in the middle by selecting the right component in the left menu, then clicking the area in the workspace to the right to place. You can then drag between contacts in order to add wiring. Using this method, you can not only place the included components (pins, gates, etc.) but also the circuits that we have added. Note how the Ripple-Adder circuit in the circuits.circ file contains several of the full-adders we designed.

If you think you have designed a cool circuit, feel free to email it to me to show off!
